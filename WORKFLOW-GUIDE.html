Phase 2 — Autonomous Build

Approve the roadmap, then let Claude build the entire milestone.

/autopilot
  • 🎯 Creates GitHub milestone + issues from roadmap
  • 🔧 Implements each issue: Data → Logic → UI → Tests
  • 📝 Logs architectural decisions as ADRs
  • ✅ Runs quality gates (lint, test, build, security)
  • 🔄 Retries failures up to 3×, skips persistent ones
  • 📦 Creates milestone PR with full summary
📝
Decisions are logged as ADRs — review them in the milestone PR.

When Production Breaks

Skip everything. Fix it fast. Write a postmortem after.

🚨 /hotfix → 🛫 /deploy → 👀 /review

Branch from main → apply the smallest possible fix → deploy immediately
Then write a postmortem and create a follow-up issue if it was a band-aid

1

Triage the incident

Claude reads the error, classifies severity and impact

/hotfix
2

Apply minimal fix

Change the fewest files possible. No refactoring. Just stop the bleeding.

automatic
3

Deploy immediately

Expedited PR with [HOTFIX] prefix, merge, push to production

/deploy
4

Postmortem & follow-up

Document what broke, why, and how to prevent it. Create a follow-up issue if needed.

automatic
