# Design References

Save screenshots of UI patterns you like here.

## How to Use

1. When you see a UI pattern you like (on Stripe, Linear, FT, NYT, etc.), screenshot it
2. Save to this folder with a descriptive name:
   - `stripe-dashboard-nav.png`
   - `linear-task-list.png`
   - `ft-article-layout.png`
3. Reference in your prompts to Claude:
   - "Build the sidebar. Match the style in `/docs/design-references/stripe-sidebar.png`"
   - "Use the table layout from `/docs/design-references/linear-table.png`"

## Recommended Sources

Based on your preferences:

- **Stripe** (stripe.com) - Dashboard, checkout flows, data tables
- **Linear** (linear.app) - Task lists, sidebars, minimal UI
- **Financial Times** (ft.com) - Typography, article layout, data presentation
- **New York Times** (nytimes.com) - Editorial design, content hierarchy
- **Anthropic** (anthropic.com) - Clean corporate, whitespace usage
- **Notion** (notion.so) - Document editing, simple controls

## Tips

- Screenshot specific components, not full pages
- Name files descriptively so Claude understands the context
- 3-5 reference images per project is usually enough
